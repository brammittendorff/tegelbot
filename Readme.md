# Tegeltjes bot for discord
 
Here you can create "tegeltjes" for discord. When saying `/tegeltje Hey it is working live`.
 
## Requirements
 
- node
- npm
 
 
## Configuration
 
```
npm install
```
 
## Usage
 
```
node index.js

```
